网站图片分布为以期分布存储。

主页地址为http://www.mmxyz.net/rosi-加上期号。

在每期内的分布为图片地址为：http://img1.mmxyz.net/（年份）/（月份）/rosi-（期号）-（图片标号）.jpg

所以直接找到地址直接下载。
